Abstract

This Note replaces a draft specification for a programmatic interface (API) to certain types of transformation matrices. See Geometry Interfaces Module Level 1 [[GEOMETRY-1]] for newer versions of that API.

Status of this document

The work on the DOMMatrix specification is continued in Geometry Interfaces Module Level 1. No further work is planned on this Note.
